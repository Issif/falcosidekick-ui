# Falcosidekick UI

* Required NodeJS >= v14

## Project setup
```
npm install
```

## Development Prerequiements

* Copy `.env.example` to `.env`
* Run the Falcosidekick UI API

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
